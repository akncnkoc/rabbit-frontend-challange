## Rabbit Front-End Challange

### Used Libs
 - Vite
 - Redux - Redux Toolkit
 - Tailwind
 - Headless UI
 - HeroIcons - React Icons
 - React Router DOM

### Features
 - Languages (TR - EN)
 - Top Rated - Trend Movies
 - Movie -> Overview,Credits, Voting, Cast&Crew, Productions
 - About

### Instructions

```sh
npm install
```

```sh
npm run dev
```
